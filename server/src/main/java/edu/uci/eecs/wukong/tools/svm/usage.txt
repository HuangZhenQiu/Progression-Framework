scaling,taining and testing:
java -cp libsvmnew.jar svm_scale -l -1 -u 1 -s range  train.txt > train.scale
java -cp libsvmnew.jar svm_scale -r range test.txt  > test.scale
java -cp libsvmnew.jar svm_train -s 0 -c 5 -t 2 -g 0.5 -e 0.1 train.scale  model_scale.txt
java -cp libsvmnew.jar svm_predict -b 1 test.scale  model_scale.txt  output_scale.txt

training and testing :
java -cp libsvmnew.jar svm_train -s 0 -c 5 -t 2 -g 0.5 -e 0.1 train.txt  model.txt
